# Stock Market Briefing Skill

全球财经早餐 - 多源数据智能分析

## 简介

这是一个综合性的股市简报技能，整合了多个数据源，包括：

- **金十数据 (Jin10)** - 实时行情和快讯
- **东方财富** - A股和港股数据
- **同花顺** - 板块和行业数据
- **雪球** - 热股和市场情绪
- **Tavily** - 美股和全球新闻

## 功能特点

### 简报内容
1. 🔥 **今日优选** - 8条重要市场新闻
2. 📊 **市场盘点** - 大宗商品、美股、港股、A股
3. 🛢️ **中东局势** - 地缘政治动态
4. 🇨🇳 **A股表现** - 市场亮点
5. ⚠️ **今日风险预警** - 经济日历
6. ⚡ **美联储动态** - 央行政策
7. 🔥 **热点板块** - 领涨板块排行

### 数据源详情

| 数据源 | 类型 | 数据量 |
|--------|------|--------|
| 金十数据 | 快讯、大宗商品 | 实时 |
| 东方财富 | A股、港股、新闻 | 全市场 |
| 同花顺 | 概念板块、行业 | 375概念+90行业 |
| 同花顺港股 | 港股通 | 4619只股票 |
| 雪球 | 热股排行 | 100只热股 |
| Tavily | 美股、全球新闻 | 实时搜索 |

## 安装

### 方式一：通过 ClawHub 安装
```bash
clawhub install stock-briefing
```

### 方式二：手动安装
```bash
# 1. 克隆仓库
git clone https://github.com/loubin9-art/liveskill.git

# 2. 复制到 OpenClaw skills 目录
cp -r liveskill/stock-briefing ~/.openclaw/workspace/skills/

# 3. 安装 Python 依赖
pip install akshare

# 4. 配置环境变量
export TAVILY_API_KEY="your-api-key"
```

## 使用

### 手动运行
```bash
node ~/.openclaw/workspace/skills/stock-briefing/scripts/briefing.mjs
```

### 定时任务
已配置自动定时任务：
- 早上 10:00（工作日）
- 下午 17:00（工作日）

## 配置

### 必需环境变量
- `TAVILY_API_KEY` - Tavily API Key（用于美股数据和新闻搜索）

### 可选配置
编辑 `scripts/briefing.mjs` 可自定义：
- 新闻数量
- 显示板块
- 数据源选择
- 输出格式
